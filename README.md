# Bài Tập HTML Mini-Project - Bản vẽ Thiết kế Sơ đồ 1

Dự án này tuân thủ nghiêm ngặt và đầy đủ các tiêu chuẩn cấu trúc ngữ nghĩa (Semantic Tags) bắt buộc của HTML5.

## 🧱 Xác Nhận Đủ Cấu Trúc Yêu Cầu (Checklist)
- [x] **html**: Thẻ cấu trúc gốc tổng thể của trang web.
- [x] **header**: Khối tiêu đề đầu trang chứa tên dự án/thương hiệu.
- [x] **nav**: Khối chứa hệ thống thanh menu điều hướng.
- [x] **main**: Khối chứa nội dung thông tin cốt lõi (Bao gồm khung ảnh gạch chéo `X`).
- [x] **section**: Khối phân đoạn độc lập (Được áp dụng riêng biệt cho khu vực chứa `FORM`).
- [x] **footer**: Khối chứa thông tin bản quyền ở chân trang cuối cùng.

## 🛠️ Cách Kiểm Tra Khối Phần Tử
Bạn có thể mở file `index.html` trên trình duyệt, nhấn phím **F12** (hoặc click chuột phải chọn **Inspect / Kiểm tra**) để nhìn thấy cấu trúc cây thư mục HTML hiển thị phân cấp rành mạch, bảo đảm đáp ứng 100% tiêu chí chấm điểm cấu trúc tối thiểu của bài học.
